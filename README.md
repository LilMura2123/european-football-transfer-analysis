# Football Transfer Market Analysis

This project analyzes football transfer strategies across major European leagues using Python and machine learning.

## Project Goal

The goal of this project is to explore patterns in football transfers and analyze how clubs behave in the transfer market.

Key questions:

- Which clubs spend the most?
- Do clubs prefer young players?
- How do transfer strategies differ across leagues?

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Structure

notebooks/transfer_market_analysis.ipynb – main analysis notebook

data/ – dataset description

images/ – visualizations
